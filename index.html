Visual Studio Code is a coding platform that allows you to build different websites. It uses 3 languages: HTML, CSS, JS. HTML (HyperText Markup Language) is the most basic building block of the Web. It defines the meaning and structure of web content. Other technologies besides HTML are generally used to describe a web page's appearance/presentation (CSS) or functionality/behavior (JavaScript). Cascading Style Sheets (CSS) is a style sheet language used for describing the presentation of a document written in a markup language such as HTML. CSS is a cornerstone technology of the World Wide Web, alongside HTML and JavaScript. JavaScript, often abbreviated as JS, is a programming language that conforms to the ECMAScript specification. The Console Screen is a hidden screen in a webpage. The Console Screen is a tool which is mainly used to log information associated with a web page like: network requests, javascript, security errors, warnings, CSS etc. It enables us to interact with a web page by executing javascript expression in the contents of the page. To open the console screen, Right click once with your mouse anywhere on a webpage, Then an option "Inspect" will appear and click it. A window will appear showing the code for the website. Click "Console" in the tabs bar on the top of the window, the console screen will appear. If you want to put anything on the console screen, use the code "console.log("write the message you want here").
